Propagation of a ray bundle.

Propagates a ray bundle calling the integrator for each photon.

Template Parameters
OrderOctree interpolation order : 0 for NGP, 1 for CIC, 2 for TSC or -1 for an homogeneous universe.
RK4Runge-kutta of fourth order or euler.
VerboseVerbose mode for debug purposes.
CosmologyCosmology evolution type.
OctreeOctree type.
TypeScalar type.
DimensionNumber of space dimension.
HomogeneousHomogeneous reference.
Pointpoint type.
Parameters
[in]photonCentral photon initial data.
[in]countNumber of other photons to use.
[in]angleHalf-angle at the cone vertex.
[in]rotationArbitrary rotation to optionally apply on the resulting circle of photons.
[in]interpolationStop condition : redshift, a, t, r.
[in]cosmologyCosmology evolution.
[in]octreeOctree.
[in]vobsObserver peculiar velocity, in SI
[in]lengthSpatial length in SI units.
[in]nstepsNumber of lambda steps per grid.
[in]aminIf different from zero, all photons should end by this value of a.
[in]filenamesFile names of the output. If empty, no output. If at least on percent sign, all trajectories are saved. Otherwise, only the central one is saved.
[in]HomogeneousOptional homogeneous trajectory. If not provided the angular diameter distance use the inhomogeneous value of a. If provided, the homogeneous value of a for the given radius is used.
Returns
Central photon trajectory.
